# 接口文档

备注：

1. 请求参数中的参数名对应的是json中的key值，value值可能是String，可能是一个对象
2. 返回参数中的{list}表示对应对象的列表，{对象名}表示一个对象



## 登录

|   URL    | /user/login                                                  |
| :------: | ------------------------------------------------------------ |
| 请求参数 | username，password                                           |
| 返回参数 | 200：{"userId":2,"nickName":"joanie","loginPassword":"123456","email":"15927444722@163.com","sex":2,"userPic":"1.1.1.1/joanie/portrait","birthday":"Oct 25, 1998 12:00:00 AM"}<br>405：{"info": "用户不存在"} |
